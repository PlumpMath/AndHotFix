命令 : fixbug.bat -f newClasses.jar -t oldClasses.jar -o patch.jar

fixbug.bat	批处理文件，主要是调用fixbugtool.jar
-f newClasses.jar	新jar文件的路径，newClasses.jar可以是绝对路径，默认是相对路径
-t oldClasses.jar	旧jar文件的路径，oldClasses.jar可以是绝对路径，默认是相对路径
-o patch.jar	输出文件，，patch.jar可以是绝对路径，默认是相对路径
